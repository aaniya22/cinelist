# CineList 🎬

A personal movie and show watchlist app built with React and the TMDB API.

## Features
- Search movies and shows in real-time
- Add titles to your personal watchlist
- Mark titles as watched
- Filter by All, Watched, and Unwatched
- Clean dark mode UI

## Tech Stack
- React.js
- TMDB API
- CSS

## Live Demo
[View Live](https://cinelist-indol.vercel.app)

## Getting Started
```bash
git clone https://github.com/aaniya22/cinelist.git
cd cinelist
npm install
npm run dev
```
